What is the output of the following code? printf("%d", (10 + 20));  
A) 30  
B) 1020  
C) Error

A) %d  
B) %lf  
C) %f  
D) %c

Which keyword is used to prevent a variable from being modified in C?  
A) final  
B) static  
C) const  
D) volatile  
ANSWER: C

A) int p;  
B) int *p;  
C) pointer int p;
